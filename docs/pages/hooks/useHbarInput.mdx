# useHbarInput Hook (Hedera-first)

The `useHbarInput` hook converts between native token and USD for a given input and display mode. It **defaults to Hedera testnet (HBAR)**. It returns both values, the current native currency price, and loading/error flags. For Ethereum-first apps, use **[useEtherInput](/hooks/useEtherInput)** (defaults to mainnet).

## Import

```tsx
import { useHbarInput } from "@scaffold-ui/hooks";
```

## Usage

```tsx
const {
  valueInEth,
  valueInUsd,
  nativeCurrencyPrice,
  isNativeCurrencyPriceLoading,
  isNativeCurrencyPriceError,
  nativeCurrencySymbol,
} = useHbarInput({ value, usdMode });
// Optional: pass chain (default hederaTestnet)
useHbarInput({ value, usdMode, chain: mainnet });
```

## Parameters

| Parameter | Type      | Default          | Description                                                                 |
| --------- | --------- | ---------------- | --------------------------------------------------------------------------- |
| `value`   | `string`  | -                | The current input value. Interpreted as native token or USD depending on `usdMode`. |
| `usdMode` | `boolean` | -                | If `true`, `value` is USD; if `false`, `value` is native token (e.g. HBAR). |
| `chain`   | `Chain`   | `hederaTestnet`  | Chain for price and symbol (HBAR for Hedera, ETH for mainnet).              |

## Return Values

| Property                       | Type     | Description                                                                 |
| ------------------------------ | -------- | --------------------------------------------------------------------------- |
| `valueInEth`                   | `string` | The input value expressed in native token. Empty string when conversion is unavailable. |
| `valueInUsd`                   | `string` | The input value expressed in USD. Empty string when conversion is unavailable. |
| `nativeCurrencyPrice`         | `number` | The fetched native currency price in USD. `0` when not yet available.       |
| `nativeCurrencySymbol`        | `string` | Symbol for the chain (e.g. `"HBAR"`, `"ETH"`).                             |
| `isNativeCurrencyPriceLoading` | `boolean`| Loading state for fetching the price.                                       |
| `isNativeCurrencyPriceError`   | `boolean`| Error state for fetching the price.                                         |

## Backward compatibility

**useEtherInput** is an alias that defaults `chain` to `mainnet` (ETH). Use `useHbarInput` for Hedera apps; use `useEtherInput` when you want ETH by default.
