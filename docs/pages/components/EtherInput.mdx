# EtherInput Component (backward-compatible alias)

The `EtherInput` component is a **backward-compatible alias** for [HbarInput](/components/HbarInput). It defaults `chain` to **mainnet (ETH)**. For Hedera-first apps, use **HbarInput** (defaults to hederaTestnet / HBAR). Same props and behavior; only the default chain differs.

## Import

```tsx
import { EtherInput } from "@scaffold-ui/components";
```

## Props

| Prop             | Type                                                                                   | Default | Description                                                                                              |
| ---------------- | -------------------------------------------------------------------------------------- | ------- | -------------------------------------------------------------------------------------------------------- |
| `name`           | `string`                                                                               | -       | Name attribute for the input element                                                                     |
| `placeholder`    | `string`                                                                               | -       | Placeholder text for the input                                                                           |
| `defaultValue`   | `string`                                                                               | -       | Initial value for the input                                                                              |
| `defaultUsdMode` | `boolean`                                                                              | `false` | Start with the input in USD mode instead of native token                                                  |
| `chain`          | `Chain`                                                                                 | `mainnet` | Chain for price and label (EtherInput defaults to ETH; use [HbarInput](/components/HbarInput) for HBAR default) |
| `disabled`       | `boolean`                                                                              | `false` | Disables the input and toggle button                                                                     |
| `onValueChange`  | `(value: { valueInEth: string; valueInUsd: string; displayUsdMode: boolean }) => void` | -       | Called when the value or display mode changes. Provides both ETH and USD values and current display mode |
| `style`          | `CSSProperties`                                                                        | -       | Custom CSS styles (memoize to prevent re-renders)                                                        |

### HBAR mode (Hedera)
